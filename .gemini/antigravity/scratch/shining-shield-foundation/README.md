# Shining Shield Foundation Website

This is the official website for the **Shining Shield Foundation**, a non-profit organization dedicated to national stability, emergency response, and community empowerment.

## Project Overview

The website is designed to convey trust, authority, and hope, tailored for a presidential-level foundation. It serves as a hub for:
- Informing the public about the foundation's mission and activities.
- Highlight impact and upcoming events.
- Facilitating donations and volunteer sign-ups.
- Providing transparency through leadership profiles and contact information.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: Inter (Body) & Poppins (Headings) via `next/font`

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view the site.

3.  **Build for Production**:
    ```bash
    npm run build
    npm start
    ```

## Project Structure

- `app/`: Contains the main application routes and `layout.tsx`.
- `components/`: Reusable UI components for each section of the landing page (Hero, About, Contact, etc.).
- `public/`: Static assets (images, icons).

## Design System

- **Primary Color**: Deep Navy (`#0a1f44`) - Represents Authority.
- **Accent Color**: Gold/Amber (`#d4af37`) - Represents Prosperty & Official status.
- **Typography**: Clean, modern sans-serif fonts for readability and professional appearance.
